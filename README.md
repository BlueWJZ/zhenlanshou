# 真藍瘦

真藍瘦是以台灣外食情境為核心的飲食、運動、飲水、睡眠與體重紀錄 App。現有版本是可部署的 Vinext／Cloudflare Worker 網頁版，個人健康紀錄預設只存於使用者裝置。

## 目前功能

- 4,195 項食品：1,878 項常見外食、2,173 項食藥署資料與 144 項單品
- 食品搜尋、文字份量解析、條碼查詢、裝置端營養標示 OCR
- 熱量、蛋白質、碳水、脂肪、鈉、糖與纖維估算
- 36 種 MET 運動消耗估算
- 體重、飲水、睡眠、常用組合與自訂食譜
- 飲食與喝水定時提醒總開關
- 7／30 天趨勢與本機教練回顧
- 深色模式、CSV 匯出、JSON 完整備份／還原與資料刪除
- 隱私權政策、使用條款、計算方法、安全標頭與錯誤復原

## 開發需求

- Node.js `>=22.13.0`
- 依 `package-lock.json` 安裝依賴

## 常用指令

- `npm run dev`：啟動本機開發環境
- `npm run lint`：程式規範檢查
- `npm run typecheck`：前端與 Worker 嚴格型別檢查
- `npm run test:unit`：核心公式、資料、解析與備份測試
- `npm test`：正式建置、部署成品驗證與完整測試
- `npm run check`：發佈前完整檢查
- `npm run data:tfda`：重新匯入食藥署資料

## 資料與隱私

- 應用程式狀態儲存在 `localStorage`，沒有帳號或雲端同步。
- 條碼查詢會將商品條碼傳送至 Open Food Facts。
- 營養標示照片由 Tesseract.js 在裝置上辨識；首次使用需下載文字模型。
- 目前沒有廣告、第三方分析、實際付款或 AI API 呼叫。
- 新增任何 SDK、帳號、雲端同步、AI、分析或付款前，必須同步更新隱私政策與商店資料聲明。

## 重要限制

本專案目前是正式網頁版，尚不是可直接提交 App Store／Google Play 的已簽章原生專案。原生上架前仍需完成本機排程通知、原生權限、分享、商店內購、真機測試、簽章與商店後台資料。細節請見正式上線檢查表。

熱量、營養與運動消耗均為估算，不能取代醫師或營養師建議。
